## 拉库即用
```
ql repo https://github.com/pingxian/elm.git "^ele" "" "USER|common|ql|sendNotify1|ele_coupon_rely"
```
环境变量：ms 设置多少秒提前抢券（单位毫秒），默认 1000 毫秒 export ms="1000"
环境变量：delayTime 请求间隔（单位毫秒）默认 300 毫秒 export delayTime="300" 
环境变量：allTime 请求总时长单位（秒）默认 20 秒 export allTime="20"
